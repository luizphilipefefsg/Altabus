# README - Sistema Altabus

## Sobre o Projeto

O projeto **Altabus** é um sistema simples desenvolvido em HTML com duas páginas principais:

* Cadastro de usuários
* Login de usuários

O objetivo do projeto é praticar conceitos básicos de formulários HTML, campos de entrada, validações e envio de dados utilizando o método `POST`.

---

## Tecnologias Utilizadas

* HTML5

---

## Funcionalidades

### Cadastro

A página de cadastro permite que o usuário informe:

* Nome completo
* E-mail
* Telefone
* Data de nascimento
* Senha
* Aceite dos termos de uso

Além disso, possui:

* Botão de cadastro
* Botão para limpar os campos

---

### Login

A página de login permite:

* Inserção de e-mail
* Inserção de senha
* Opção “Lembrar de mim”

Também possui:

* Botão de entrar
* Botão de limpar

---
